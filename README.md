# TRPG 法術資料自動萃取腳本 (TRPG Spell Data Extractor)

這是一個自動化的 Python 腳本，專門用來從 Google Docs 讀取 D&D/Pathfinder 的法術詳細描述，並依據特定規則將萃取出來的屬性資料（成份、時間、距離、豁免等）自動寫回對應的 Google Sheets 中。

## 功能核心介紹

本腳本主要協助處理大量 TRPG 法術資料轉移的繁瑣工作：
1. **Google Drive / Docs / Sheets API 整合**：自動登入並尋找特定的目標試算表與關聯文件。
2. **精準法術區塊切分**：就算多個法術寫在同一個 Google Doc 當中，程式也能自動依照試算表給予的英文名稱 `[spells]` 進行精準定位，避免不同法術的屬性互相干擾。
3. **資料正規化與本地化**：在資料寫入試算表前，會自動將簡體文字轉換為**繁體中文（臺灣用字）**，並且將**全形括號「（）」自動轉換為半形「()」**，保持版面整潔一致。
4. **智慧屬性映射**：
   - **法術成份**：自動判斷並將 `[語言V]`, `[姿勢S]`, `[材料M]`, `[器材F]`, `[法器DF]` 簡化並分類填寫對應字元。
   - **時間與距離**：將「一个标准动作」自動對應為「標動」，將「近距/中距/遠距/接觸」之外的距離標記為「其他」。
   - **長度防禦網**：嚴格限制 `[持續時間]` 與 `[豁免說明]` 的字串長度上限（16 個字元），以及精準抓取 `[豁免]` (意志/強靭/敏捷/無) 與 `[法抗]` (可/不可) 關鍵字，防止試算表版面崩潰。
5. **不覆蓋手動資料**：只會針對**有缺漏資料**的列進行萃取與更新，確保您手動修改過的心血不會被機器人覆蓋。

## 環境需求

- Python 3.10+
- [uv](https://github.com/astral-sh/uv) (極速 Python 套件管理器)
- Google Cloud Platform (GCP) 的 API 憑證 (`credentials.json`)

## 安裝與設定

1. **安裝環境與依賴套件**:
   ```bash
   uv sync
   ```

2. **放置 GCP 授權憑證**:
   - 請至 Google Cloud Console 申請 Desktop App 的 OAuth 2.0 用戶端 ID。
   - 將下載下來的 `credentials.json` 放在專案根目錄下。

3. **環境變數設定**:
   - 複製 `.env.example` 並重新命名為 `.env`。
   - 若有需要，可以修改 `.env` 中的目標資料夾或試算表名稱。
     ```env
     TARGET_FOLDER_PATH="我的雲端硬碟/5團資料_大叔"
     TARGET_SHEET_NAME="法術詳述列表_大叔"
     ```

## 使用方式

執行核心爬蟲與寫入腳本：

```bash
uv run app.py --start-row <起始列數> [--batch-size <批次處理數量>]
```

- `--start-row`: (必填) 指定從 Google Sheet 的第幾列開始掃描與執行。
- `--batch-size`: (選填) 每次執行的最大列數限制，預設為 `1`。如果想一次處理 100 列，可以設定為 `--batch-size 100`。

### 執行範例

針對第 4 列進行單筆更新與測試：
```bash
uv run app.py --start-row 4
```

一次批次處理從第 10 列開始往下的 50 個法術：
```bash
uv run app.py --start-row 10 --batch-size 50
```

## 注意事項與授權

當您**第一次執行**時，程式會自動開啟瀏覽器要求您登入 Google 帳號進行授權。授權成功後，會在本地端產生一個 `token.json` 檔案。未來的執行將會自動讀取此憑證，不需再次登入。
